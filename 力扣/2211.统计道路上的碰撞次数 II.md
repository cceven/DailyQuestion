[2211](https://leetcode.cn/problems/count-collisions-on-a-road/submissions/682767912/?envType=daily-question&envId=2025-12-04)：看一下题目就会发现，每一辆车只要发生了碰撞就会并且只会提供1次碰撞次数，所以我们只需要求哪些车辆会发生碰撞就可以了。我们需要分别找到左边界和右边界，左边界就是第一辆不往左边跑的车，右边界就是从后往前第一辆不往右跑的车。左右边界之间的L和R一定会发生碰撞，我们统计这些车辆的数量就是我们的结果。
```cpp
class Solution {
public:
    int countCollisions(string directions) {
        int n = directions.size();
        int left = 0, right = n - 1;

        // 找到第一个不往左边跑的车
        while (left < n && directions[left] == 'L')
            left++;

        // 从后往前，找到第一个不往右边跑的车
        while (right >= 0 && directions[right] == 'R')
            right--;

        // 中间的L和R就是碰撞数量
        int ans = 0;
        for (int i = left; i <= right; i++) {
            if (directions[i] != 'S')
                ans++;
        }
        return ans;
    }
};
```
