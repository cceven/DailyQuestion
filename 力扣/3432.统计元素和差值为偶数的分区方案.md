[3432](https://leetcode.cn/problems/count-partitions-with-even-sum-difference/submissions/682905425/?envType=daily-question&envId=2025-12-05)
## 方法一
直接硬算。
```cpp
class Solution {
public:
    int countPartitions(vector<int>& nums) {
        int n = nums.size();
        int sum = accumulate(nums.begin(), nums.end(), 0);
        int left = 0, right = sum, ans = 0;
        for (int i = 0; i < n - 1; i++) {
            left += nums[i], right -= nums[i];
            if ((right - left) % 2 == 0)
                ans++;
        }
        return ans;
    }
};
```
## 方法二
方法二就是运用到了数学公式。$left + right = sum$，我们要求$right - left$的奇偶性，那么可以转换一下。$(right - left) = sum - 2 \times left$。由于$2\times left$一定是偶数，所以其实我们的结果就是看$sum$的奇偶性。只要$sum$是偶数那结果就是所有可能的组合$n - 1$，如果$sum$是奇数那结果就是$0$。
```cpp
class Solution {
public:
    int countPartitions(vector<int>& nums) {
        int n = nums.size();
        int sum = accumulate(nums.begin(), nums.end(), 0);
        if (sum & 1)
            return 0;
        else
            return n - 1;
    }
};
```
