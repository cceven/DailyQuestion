#二分答案
[875](https://leetcode.cn/problems/koko-eating-bananas/description/)：当每小时吃k个能吃完时，每小时吃k+1个一定吃不完。当每小时吃k个吃不完时，每小时吃k-1个一定吃不完。符合**二分答案**类题目的做法。
首先考虑极端情况，当h无限大时，我们最小可以取1即下界。据题目`piles.length <= h <= 109`可以知h最小为`piles.length`，此时如果要吃完那么k需要等于piles中最大的一堆的数目即上界。
判断当前k能否吃完，直接使用向上取整一个一个累加每一堆需要花费的时间即可。
```cpp
class Solution {
public:
    int minEatingSpeed(vector<int>& piles, int h) {
        long long left = 1, right = *max_element(piles.begin(), piles.end());
        long long ans = 0;
        while (left <= right) {
            long long mid = left + (right - left) / 2;
            if (check(piles, h, mid)) {
                ans = mid;
                right = mid - 1;
            } else {
                left = mid + 1;
            }
        }
        return ans;
    }

private:
    bool check(vector<int>& piles, int h, int k) {
        long long use_t = 0;
        for (long long p : piles)
            use_t += (p + k - 1) / k;

        return use_t <= h;
    }
};
```
